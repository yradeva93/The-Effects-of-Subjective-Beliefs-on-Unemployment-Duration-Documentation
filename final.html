Visualisation and Results Formatting¶

The directory src/final contains code which uses the already created data from the analysis in src/analysis and generates the needed tables and figures.

Table 1 - Unemployment Duration¶

The file “table_unemployment_duration.do” creates a table with the estimates of subjective probability of finding a job for unemployment duration from the corresponding do-file in the analysis folder “unemployment_duration.do” It writes the results to Latex file “`”${PATH_OUT_TABLES}/table_unemployment_duration.tex”’”

Table 2 - Number of Applications¶

The file “table_num_applications.do” creates a table with the estimates of subjective probability of finding a job for number of applications from the corresponding do-file in the analysis folder “num_applications.do” It writes the results to Latex file “`”${PATH_OUT_TABLES}/table_num_applications.tex”’”

Table 3 - Number of Search Channels¶

The file “table_num_channels.do” creates a table with the estimates of subjective probability of finding a job for number of search channels from the corresponding do-file in the analysis folder “num_channels.do” It writes the results to Latex file “`”${PATH_OUT_TABLES}/table_num_channels.tex”’”

Classification Figures¶

The module “classifciation_figures.py” creates two figures based on “classification.csv” and “classification_cleaned.csv”. The file “classification.csv” consists of 2 columns (‘importance’ and ‘feature’). In a bar chart (“feature_importances.png”) the 20 most important features are listed and their predictive power for finding a job is shown. The other figure (“subj_prob_finding_factorplot.png”) is based on “classification_cleaned.csv”. It plots the unemployment duration against the subjective probability of finding a job.

Classification Accuracy¶

This module reads in the “classification_accuracy.pickle” file and saves the data as a Latex file that can be later included in the paper.
